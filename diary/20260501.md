|---|---|---|---|---|---|---|---|---|---|

### 重要な発見

1. **9B admission > 4B admission** が定量で確認 — drug recall +35% 相対改善。**2026-04-23 に 9B 専用化した本番判断はベンチマークでも裏付けられた**
2. **音声 STT 形式は構造化の半分以下のスコア** (4B SOAP の場合 ROUGE-L 0.140 vs 構造化 0.383)
3. **production parser に追加すべきマーカー**: `【S:`(半角コロン)と`【S - Subjective(...)`(parenthesized) を追加すれば parse 失敗 -27%
4. **共通 missed パターン**: ビソプロロール、Killip分類、NYHA、CCS分類等のスコアリング詳細
5. **4B SOAP のハルシネーション** が voice 形式で顕在化 (大動脈解離で常用薬を Plan に書き戻す等) → RAG or 9B SOAP 切替で改善可能性

## 5. 専門医レビューから得た主な教訓

### カルテ表現の慣習 (do/don't)

| ❌ 書かない | ✅ 代替 |
|---|---|
| 「即時 12 誘導 ECG」 | 「12 誘導 ECG」 |
| SpO2 95% を「軽度低下」 | 数値のみ (95% は正常範囲) |
| BP 108/68 を「やや低め」 | 削除 (正常範囲) |
| 再診で「活気良好」「全身状態」 | 削除 |
| 「強疑時」 | 「を強く疑う場合」 |
| 「家庭でのコンディショニング維持」 | 「自宅でも運動習慣は維持されている」 |
| 「元喫煙者」 | 「過去喫煙」「past smoker」 |
| 「持続的に出現」 | 「持続」 |

### 患者向け会話表現

| ❌ 言わない | ✅ 自然な言い方 |
|---|---|
| 「12 誘導心電図」 | 「心電図」 |
| 「簡単な聴診」 | 「胸の音を聞かせてもらいます」 |
| 「採血もお願いします」 | 「採血もさせていただきます」 |
| 「診察します」 | 「お話をお聞かせください」 |
| 「早死にした親族」 | 「若くして亡くなられた親族の方」 |

### 検査・処置オーダーの現実度

- atypical 症状で **トロポニン 0/3/6 時間連続** → 1 回採血、陽性なら追加
- **SpO2 経時モニター** はルーチンで書かない
- 安定狭心症で **3 ヶ月毎心エコー不要** (有症候時 or 6-12 ヶ月毎)
- 普段の再診で **橈骨動脈・足背動脈の触診はしない**
- **V3R/V4R 右側胸部誘導はルーチンではない**
- **N-アセチルシステインは現状使われない** (造影剤腎症予防として)
- ECG で明らかな ST 変化があれば **採血を待たず搬送**
- 心機能良好な人に **β遮断薬・ACE 阻害薬は不要** (慣行的継続は OK)

### 構造的問題: Plan の冗長性

- 専門医「**日常でここまでカルテは書けない**」「**Plan は Plan だけを書くべき**」
- 修正方針: Plan **5-8 項目以内**、Assessment と内容重複させない、A で書いた病態解説を P で繰り返さない

### 家族歴の定義

- **配偶者は家族歴ではない** (血縁ではない) → 社会歴に書く
- 「父: 戦死」など医学的に意味のない情報は不要

詳細: `data/aci_jp_cardio/outpatient/LESSONS_LEARNED.md`
