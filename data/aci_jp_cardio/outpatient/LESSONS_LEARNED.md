## 1. カルテ表現の慣習 (do/don't)

### Don't (実臨床のカルテで書かない表現)

| ❌ 書かないでほしい | ✅ 代替 / 削除 | 出典 |
|---|---|---|
| 「**即時** 12 誘導 ECG を記録」 | 「12 誘導 ECG を記録」(「即時」を削除) | [p] |
| 「**強疑時**」 | 「**を強く疑う場合**」 | [s] |
| SpO2 95% を「**軽度低下**」 | 95% は正常範囲、「低下」と書かない | [al] |
| BP 108/68 を「**やや低め**」 | 正常範囲なのでカルテに書かない (心の中で考えるのみ) | [be] |
| 再診時の「**活気良好**」「**全身状態**」「**苦悶様顔貌なし**」 | 普段の再診で書かない、削除 | [v][w] |
| 「**家庭でのコンディショニング維持**」 | 「**自宅でも運動習慣は維持されている**」 | [af] |
| 「**生涯非喫煙**」 | 「**喫煙歴: なし**」 | [ba] |
| 「**元喫煙者**」 | 「**過去喫煙 / past smoker**」 | [d][h] |
| 「**持続的に出現**」 | 「**持続**」 | [j] |
| 「**古典的**労作性狭心症」 | "古典的" を削除し「労作性狭心症としては非典型」 | [o] |

### 主訴・副主訴の言い換え

| ❌ | ✅ | 出典 |
|---|---|---|
| 「**心拍速感**」 | 「**動悸**」 | [c][m] |
| 「**急に息ができない**」 | 「**急性発症の呼吸苦**」 | [au] |
| 主訴として専門的すぎる病名 (例: ATTR-CA red flag) | 患者の言葉で記述 (息切れ、しびれ等)、紹介状なら専門病名可 | [bd] |

### 数値の解釈・記載

| 数値 | 注意 | 出典 |
|---|---|---|
| BNP 32 | 正常上限 16-18 を超えるが医学的には問題なし。**カルテシステムで「高値」赤文字表示**になることは多い (それを記載で表現) | [ac] |
| HR 56 | 定義上は徐脈だが、**「徐脈」と明確に書くほどではない**、「**やや徐脈**」が現実 | [ak] |

## 2. 患者向け会話表現 (Pattern B 口語対話)

### 患者に向けて言わない表現

| ❌ 言わない | ✅ 自然な言い方 | 出典 |
|---|---|---|
| 「**12 誘導心電図**」 | 「**心電図**」「心電図検査」 | [f] |
| 「**簡単な聴診**」 | 「**胸の音を聞かせてもらいます**」 | [e] |
| 「採血**もお願いします**」 | 「採血**もさせていただきます / 取らせていただきます**」 | [g] |
| 「**診察します**」 | 「(具体的な行動)」、「お話を伺ってもよろしいですか」 | [av] |
| 「**早死にした親族**」 | 「**若くして亡くなられた親族の方**」 | [bg] |

### 説明の粒度

- **問診段階で詳細病態説明はしない**。「**これから検査をしていきます。検査結果を見てまたご説明させていただきますね**」が自然 [bc]
- 患者・家族が **専門スコア (MMSE 23 点、CHA2DS2-VASc 等) を口で説明することはない**。家族の発言として書くなら「物忘れの検査で軽度の認知症と言われた」程度 [bb]

## 3. 検査・処置オーダーの現実度

### 過剰オーダーを避ける

| ❌ 過剰 | ✅ 現実的 | 出典 |
|---|---|---|
| atypical 胸痛で **トロポニン 0/3/6 時間連続** | **採血は 1 回**。陽性なら数時間後追加。連続前提で書かない | [q] |
| **SpO2 経時モニター** (atypical 胸痛で) | 必要時のみ、ルーチン記載しない | [r] |
| 安定狭心症で **3 ヶ月毎の心エコー** | 基本不要。**有症候時、または間隔を空けてフォロー** | [y][ag] |
| 普段の再診で **橈骨動脈・足背動脈の触診** | **再診ではまずしない**、削除 | [x][aa][ab] |
| 急性下壁梗塞ルーチンで **V3R/V4R 右側胸部誘導** | ルーチンではない、**右室梗塞疑いで普通の心電図に所見がない時のみ** | [am] |
| 造影剤腎症予防に **N-アセチルシステイン** | **使われていない** (腎保護効果が証明された薬・点滴は現状なし) | [ap] |
| ECG で **明らかな ST 変化があるのに採血を待つ** | **採血を待たず搬送** (door-to-balloon 短縮) | [ao] |
| 心機能良好な人 (壁運動異常なし) に **β遮断薬・ACE 阻害薬** | **必要ない**、SOAP の P から削除 | [ad] |
| HbA1c 改善傾向で **SGLT2-i 追加検討** | 改善なら不要。**HbA1c 上昇傾向時に追加検討** が現実 | [ae] |
| たこつぼ鑑別で **トロポニン連続フォロー** | ACS でもタコツボでも上昇するので **連続フォローの必要は薄い** | [ay] |

### 患者管理の常識

- ARB / ACE 阻害薬を内服中で **立ちくらみ + BP 低下傾向** あれば **減量 (5→2.5mg) または中止** を指示する [at]
- 肺炎合併の心不全では **痰の喀出ができず悪化リスク**を P に含める価値あり [aw]
- 禁煙して 1 年弱の患者に **再度強い禁煙指導は不要** (してもいいが冗長) [t]

## 4. SOAP の構造 (重要)

### Plan の冗長性問題

専門医から **2 ケース** ([ax] AHF-B, [bf] AS-B) で**「Plan が長すぎ、Assessment と内容が被っている」**と指摘あり。

**「日常でここまでカルテは書けない」「あくまで Plan は Plan だけを書くべき場所」**

### 修正方針

- Plan は **5-8 項目以内** が現実的、**10 項目超は明らかに過多**
- A (Assessment) で書いた診断根拠・病態評価を **P で重複させない**
- P には **次に行う具体的アクション (検査・処置・薬剤・教育・次回再診) のみ** を記述
- 「Red flag 説明」「次回再診タイミング」は P 末尾に簡潔に
- 説明文中に教育的な解説 (病態のメカニズム、ガイドライン引用) を入れない (それは A の役割)

### A (Assessment) と P (Plan) の役割分担

| section | 書くこと | 書かないこと |
|---|---|---|
| **A** | 診断 (確定 or 鑑別)、根拠、リスク評価、合併症、must-not-miss | 治療計画、検査オーダー |
| **P** | **次のアクション**: 検査・処置・薬剤・教育・搬送 / 紹介・再診タイミング | 病態の解説、根拠の繰り返し、ガイドライン引用 |

## 5. 家族歴・社会歴の定義

### 家族歴 (3 親等以内が原則)

- **配偶者 (夫・妻) は家族歴ではない** (血縁ではない) [ah][aj]
- 配偶者の既往は **社会歴**として「妻が認知症で介護中」のように記載
- 父母・兄弟姉妹・子・祖父母・孫・叔父叔母・甥姪が家族歴

### 社会歴

- **「父: 戦死」など、医学的に意味のない情報は不要** [az]
- 喫煙: 「なし」/「過去喫煙 (xx pack-year)」/「現喫煙 (xx 本/日)」
- 飲酒: 「なし」/「機会飲酒」/「常習 (xx ドリンク/日)」
- 職業・生活様式・家族構成・介護状況などはここに

## 6. 患者プロファイル (初診時)

### 持っていない情報

- **血液型は初診で持っていることはほぼない** [b]。`patient.blood_type` は **null** がデフォルト、特殊状況 (献血手帳持参、輸血歴で本人記憶) のみ記載
- お薬手帳がない場合は患者口頭で「血圧の薬」のような曖昧記載でリアル

### 文章の分け方

- 社会歴の中で **「デスクワーク」と「ジョギング 30 分/日」**を**同じ文に並べない**。職業と運動習慣で文を分ける [i]
  - ❌「デスクワーク (会社員)、ジョギング 30 分以上/日」
  - ✅「仕事はデスクワーク中心。運動習慣としてジョギングを 30 分/日継続」

### 細かい表現

- 「ジョギング**中**増悪なし」→「ジョギング**中の**増悪なし」(助詞「の」を補う) [n]

## 7. 既往歴の質問

- 「**他に何か病気をされたことはありますか？**」が自然な聞き方 [ai]
- 「過去に大きな病気は」「他に持病は」など、自然な日本語で
